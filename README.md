# 💍 Cha de Panela da Emily

Site de confirmação de presença e lista de presentes para o chá de panela da Emily.

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
- Firebase (Firestore)

## 📁 Estrutura do Projeto

```
cha-de-panela-Emily/
├── index.html          ← Tela 1 (boas-vindas da Emily)
├── presentes.html      ← Tela 2 e 3 (lista de presentes e modal de reserva)
├── admin.html          ← Tela 4 (painel admin)
├── firestore.rules     ← Regras de Segurança do Firebase
├── css/
│   └── style.css       ← Estilos globais
├── js/
│   ├── firebase-config.js  ← Configuração e inicialização do Firebase
│   ├── app.js              ← Lógica da página principal e de presentes
│   └── admin.js            ← Lógica do painel admin (CRUD e gerenciamento)
```
