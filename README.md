# 💍 Cha de Panela da Emily

Site de confirmação de presença para o chá de panela da Emily.

## 🚀 Funcionalidades

- Confirmação de presença
- Envio de mensagem para os noivos
- Lista de presentes
- Contagem regressiva
- Galeria de fotos
- Página de recados

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
- Firebase

## 📋 Pré-requisitos

- Node.js
- npm
- Firebase CLI

## 📦 Instalação

```bash
npm install
```

## 🚀 Execução

```bash
npm run dev
```

## 🚀 Deploy

```bash
npm run deploy
```
